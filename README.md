
# Project Title

Healthcare Patient Outcomes & Cost Efficiency Dashboard

# Project Overview

This project analyzes healthcare patient outcomes and treatment costs.

# Objectives
1. Analyze recovery rates across departments.
2. Examine readmission risk by age group.
3. Evaluate what type of treatment incurs the highest average costs.
4. Assess department cost in relations to patient outcome.
5. Determine how treatment costs vary across different age groups and regions.
6. Investigate if there's a correlation between the length of stay and the recovery score amongs patients.

# Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Scatter Plots

# Key Findings
1. Neurology department recorded the lowest recovery score.
2. Patients aged 60+ had the highest readmission risk.
3. Surgery had the highest average treatment cost.
4. Certain departments achieved better outcomes at lower costs.
5. Patients aged 60+ incurred the highest treatment costs in the North Region.
6. There's no significant correlation between how long a patieng stays in the hospital on their recovery.


# Recommendations
- Improve interventions in lower-performing departments.
- Focus readmission reduction strategies on older patients.
- Investigate drivers of high treatment costs.
- Hospital marketing strategy should focus on regions and coresponding age groups that performed poorly.
